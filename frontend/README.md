# Adventure-sv Frontend

Frontend React/Vite para generar y visualizar itinerarios turisticos.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Variables

Copiar `.env.example` a `.env`.

```txt
VITE_API_URL=http://localhost:4000
VITE_USE_MOCKS=true
```

Con `VITE_USE_MOCKS=true`, la app responde con datos locales si el backend aun no esta listo.
